单个神经元的 Logistic Regression

Logistic Regression

$$ z^{(i)} = w^T x^{(i)} + b \tag{1} $$

$$ \hat{y}^{(i)} = a^{(i)} = sigmoid(z^{(i)})\tag{2} $$

$$ \mathcal{L}(a^{(i)}, y^{(i)}) = - y^{(i)} \log(a^{(i)}) - (1-y^{(i)} ) \log(1-a^{(i)})\tag{3} $$

成本函数： $$ J = \frac{1}{m} \sum_{i=1}^m \mathcal{L}(a^{(i)}, y^{(i)})\tag{4} $$

激活函数采用 sigmoid 函数：

$$ sigmoid( w^T x + b) = \frac{1}{1 + e^{-(w^T x + b)}}\tag{5} $$

def sigmoid(z):
    s = 1/(1+np.exp(-z))

前向和后向传播

  1. 输入 X

  2. 计算： $A = \sigma(w^T X + b) = (a^{(1)}, a^{(2)}, …, a^{(m-1)}, a^{(m)})$

  3. 计算成本函数：$J = -\frac{1}{m}\sum_{i=1}^{m}y^{(i)}\log(a^{(i)})+(1-y^{(i)})\log(1-a^{(i)})$

  4. 计算偏导数： $$ \frac{\partial J}{\partial w} = \frac{1}{m}X(A-Y)^T\tag{6} $$

$$ \frac{\partial J}{\partial b} = \frac{1}{m} \sum_{i=1}^m (a^{(i)}-y^{(i)})\tag{7} $$

def propagate(w, b, X, Y):
    m = X.shape[1]
    
    # FORWARD PROPAGATION (FROM X TO COST)
    # compute activation
    A = sigmoid(np.dot(w.T, X)+b)                                    
    # compute cost
    cost = -np.sum((np.multiply(Y, np.log(A))+np.multiply(1-Y,np.log(1-A))))/m                                
    # BACKWARD PROPAGATION
    dw = np.dot(X, (A-Y).T)/m
    db = np.sum(A-Y)/m
    
    grads = {"dw": dw,
             "db": db}
    
    return grads, cost

def optimize(w, b, X, Y, num_iterations, learning_rate, print_cost = False):
    costs = []
    for i in range(num_iterations):
        # Cost and gradient calculation
        grads, cost = propagate(w, b, X, Y)
   
        dw = grads["dw"]
        db = grads["db"]
        
        # update
        w = w-learning_rate*dw
        b = b-learning_rate*db
        
        if i % 100 == 0:
            costs.append(cost)
        
        # Print the cost every 100 training iterations
        if print_cost and i % 100 == 0:
            print ("Cost after iteration %i: %f" %(i, cost))
    
    params = {"w": w,
              "b": b}
    
    grads = {"dw": dw,
             "db": db}
    
    return params, grads, costs

预测结果

  1. 计算 $\hat{Y} = A = \sigma(w^T X + b)$
  2. 将 a 转化为预测结果 1（if activation > 0.5）或者 0（if activation <= 0.5）
